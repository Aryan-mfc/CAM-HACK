### Installation On Termux
 
 
```bash
$ apt update
$ apt upgrade
$ pkg install git
$ pkg install python
$ pkg install python2
$ pkg install figlet
$ git clone https://github.com/Aryan-mfc/CAM-HACK.git
$ cd CAM-HACK
$ ls
$ chmod +x *
$ python CamHack.py

 Tool Installation successful

```
